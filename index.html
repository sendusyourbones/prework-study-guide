HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the webpage, which is important for SEO.
  • Semantic HTML elements such as body describe the content and the role it plays on a webpage.
  • Non-semantic elements such as div create a division on the page, or span creates an inline division. It is best to use semantic when possible.

CSS

  • When using an external style sheet, put a link element in the head element
  • Declaration ex. color: blue;
  • display: block; puts the the element on its own line

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git revert sha: reverts to the version identified by the sha

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a dynamic language with dynamic types. Variables are not directly associated with any particular value type, and any variable can be assigned (and re-assigned) values of all types.
  • Method: a set of instructions that can be executed by a computer.
